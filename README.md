贪吃蛇
======

***
写在前面<br/>
记得第一次做贪吃蛇，是在大一学习C++，在贴吧看到了源码，心里一动，连抄带蒙，用VS实现了出来。
晃眼间十年过去了，看到Google官方的demo，一时兴起又写了一遍。当时写了一周的贪吃蛇，今天只写了不到两小时。那时候没想到自己以后会当程序员吧。

***
最简单的玩法<br/>
* 蛇的移动
* 苹果随机出现
* 根据Touch位置改变方向
* 蛇吃苹果后变长
* 游戏失败条件
* too young, sometimes naive
